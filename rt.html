RealTransparent

RealTransparent is an application that organizes listings for property managers and allows them to access it from anywhere in real-time. Building this gave me valuable experience working with back-end developers and taught me the importance of following a good workflow. This app uses the Ruby on Rails framework and I developed the front-end using HTML5, CSS3, SCSS, JavaScript and jQuery.
